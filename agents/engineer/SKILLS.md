# Engineer — Skills

## Required skills

Load all three skills on every run.

| Skill | Purpose |
|---|---|
| `github-config` | Repository, base branch, PR template, assignees, labels, size limits |
| `fault-report-format` | Fault report structure — defines every field the Engineer consumes |
| `replay-config` | Client context, app framework, code conventions to follow when writing the fix |
