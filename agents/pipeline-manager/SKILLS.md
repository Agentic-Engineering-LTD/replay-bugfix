# Pipeline Manager — Skills

## Required skills

Load all four skills on every run.

| Skill | Purpose |
|---|---|
| `replay-config` | Client identity, Replay API key, workspace ID, project ID, severity thresholds, trigger configuration |
| `github-config` | Repository details, branch strategy, PR configuration, assignees, notification settings |
| `fault-report-format` | Fault report structure and validation rules — used to validate Analyst output before approval gate |
| `replay-mcp-tools` | Reference only — used to understand Analyst capabilities and interpret fault reports |
